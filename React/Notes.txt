React : React  is a javascript library build by meta to build fast and interactive UI 
        react is a single page application library which means it only update the part of the page which is changed and not the whole page (page does not reload only page contain changes )
        eg : facebook , instagram , twitter etc are build using react

    Why do we need react:
    javascript (b4 react )
      full page reload 
      complex DOm manipulation
      hard to maintain code
      hard to reuse code

    react (after react)
      update only part of the page which is changed
      use reusable components
      maintainable code

    What is virtual DOM:
          it creats a  copy of the real DOM  is is called virtual DOM 
          when we make any changes it only updates the parts in the virtual DOM and then it compares the virtual DOM with the real DOM and updates only the parts which are changed in the real DOM this process is called reconciliation
